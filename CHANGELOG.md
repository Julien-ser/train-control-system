# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-13

### Added
- Initial release of Train Control System
- Track network modeling with blocks and connections
- Train dynamics simulation (position, speed, acceleration, braking)
- Signal system with aspects (RED, YELLOW, GREEN, FLASHING_YELLOW, FLASHING_GREEN)
- Speed enforcement with dynamic braking curves
- Safety monitoring with automatic emergency braking
- Movement authority management
- Comprehensive test suite (54 tests)
- User guide, API reference, and architecture documentation
- Example simulation script
